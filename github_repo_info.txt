===================================================
  Sensor Data Fields
===================================================

  temperature  : 15.0 ~ 40.0  (°C)
  humidity     : 20.0 ~ 90.0  (%)
  pressure     : 980.0 ~ 1025.0 (hPa)
  light_level  : 0.0 ~ 1000.0 (lux)

  Insert interval : 2 seconds (injector.py)
  Node-RED refresh: 2 seconds
  Grafana refresh : 5 seconds
